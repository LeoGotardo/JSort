Informações dos Algoritmos

Algoritmos Básicos:

BubbleSort: O(n²) Compara elementos adjacentes e os troca se estiverem fora de ordem.

InsertionSort: O(n²) Insere cada elemento na posição correta da parte já ordenada.

SelectionSort: O(n²) Seleciona o menor elemento e o coloca na posição correta.

Algoritmos Eficientes:

QuickSort: O(n log n) Divide o array em partições menores usando um pivô.

MergeSort: O(n log n) Divide o array pela metade e mescla as partes ordenadas.

HeapSort: O(n log n) Usa uma estrutura de heap para ordenar.

Algoritmos Especializados:

ShellSort: O(n log n) Versão melhorada do insertion sort com gaps.

CountSort: O(k+n) Conta ocorrências de cada elemento (apenas números inteiros).

BucketSort: O(n) Distribui elementos em buckets e ordena cada bucket.

Algoritmo Ineficiente:

BogoSort: O(n!) Embaralha aleatoriamente até que esteja ordenado.
